#  Task 2: Phishing Awareness Training  

---

##  Project Description
An interactive phishing awareness website was developed using **HTML5, CSS3, and JavaScript**
in **Kali Linux (WSL)**.  
The project focuses on educating users about phishing attacks, how to identify phishing emails
and fake websites, and the social engineering techniques commonly used by attackers.

The website includes real-world phishing examples, visual illustrations, and an interactive quiz
with dynamic validation. Each quiz question provides instant feedback to improve user engagement
and understanding.  
This project is designed strictly for **cybersecurity education and awareness purposes**.

---

##  How to Run
This project was developed and tested using **Kali Linux on Windows Subsystem for Linux (WSL)**.

1. Clone or download the repository to your system  
2. Open **Kali Linux (WSL)**  
3. Navigate to the project directory:
   cd phishing-awareness-training

4. Open the `index.html` file to launch the website:

   Using Linux command:
   xdg-open index.html
   
  5.The phishing awareness website will load locally, allowing users to read the awareness content and attempt the interactive quizzes.
